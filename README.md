**Actividad: Galería de Fotos Interactiva y Responsiva**

Diseñar una galería de fotos responsiva que no sólo se adapte a
diferentes tamaños de pantalla, sino que también incluya interacciones
dinámicas mediante propiedades de CSS como transition, transform y
box-shadow. Los elementos deben cambiar de apariencia cuando el usuario
interactúe con ellos, mejorando la experiencia del usuario.

**Estructura de la página:**

1.  Crear una página HTML5 con:

    1.  Un encabezamiento con el título de la galería y un menú de
        navegación.

    2.  Una sección principal con **8 imágenes** dispuestas en una
        cuadrícula.

        1.  Cada imagen debe incluir un título y una descripción breve.

    3.  Un pie de página con créditos o enlaces ficticios en redes
        sociales.

**Estilo básico:**

1.  Utilizar CSS3 para:

    1.  Crear una cuadrícula de imágenes que se adapte a **escritorio,
        tableta y móvil**.

    2.  Aplicar estilos básicos como márgenes, rellenos y tipografía
        agradable.

**Interactividad con CSS:**

1.  Aplicar efectos interactivos **al pasar el cursor por encima de las
    imágenes**:

    1.  **Transformaciones (transform)**: Las imágenes deben escalar
        ligeramente.

    2.  **Transiciones (transition)**: Los cambios de estilo deben ser
        suaves.

    3.  **Sombras (box-shadow)**: Las imágenes deben tener una sombra
        que se intensifique cuando se interactúa.

2.  Mostrar un **texto oculto** sobre la imagen con un efecto de
    desplazamiento o desvanecimiento.

**Adaptación con media querías:**

1.  **Escritorio (min-width: 1200px):**

    1.  Imágenes dispuestas en una cuadrícula de **4 columnas**.

2.  **Tableta (768px a 1199px):**

    1.  Cuadrícula de **2 columnas**.

3.  **Móvil (max-width: 767px):**

    1.  Cuadrícula de **1 columna**.
